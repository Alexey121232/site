
# School Connect WebApp

## Как использовать

1. Загрузите все файлы на ваш сервер или откройте index.html локально.
2. Убедитесь, что таблица Google Sheets расшарена для сервисного аккаунта (email из credentials.json).
3. Настройте сервер на Node.js, Python или другой backend для обработки Google Sheets API через credentials.json.
4. Подключите обработку логина, создания пользователей и логирования действий.

Фронтенд — на HTML/CSS с Tailwind, логика — в script.js.
